# Code of Conduct

Safe Agent Box is a small public method project. Participation should stay practical, respectful, and focused on improving bounded AI-agent workflows.

## Expected behavior

- Keep feedback specific.
- Discuss methods and examples, not people.
- Do not share private data.
- Do not pressure maintainers to run unsafe actions.
- Use clear readback when proposing changes.
- State risks when suggesting higher permission levels.

## Unacceptable behavior

- Sharing private data.
- Encouraging destructive commands without Human Gate.
- Publishing instructions that create immediate risk.
- Harassment or personal attacks.
- Spam or unrelated promotion.

## Maintainer response

Maintainers may edit, close, or remove content that creates security risk, exposes private data, or pulls the project away from its public method scope.
