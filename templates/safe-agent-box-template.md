# Safe Agent Box Template

```text
Agent Box Name:

Project / Repo:

Goal:

Current B:

Workspace:

Permission Level:

Allowed paths:

Blocked paths:

Allowed commands:

Blocked commands:

Secrets policy:

Dependency policy:

Human Gate triggers:

Readback required:

Rollback plan:

Next checkpoint:
```

## Suggested defaults

```text
Permission Level: Level 1 or Level 2
Workspace: copy / branch / sandbox / test folder
Secrets policy: keep secrets outside chat and outside agent-readable workspace
Readback required: files changed, commands run, checks run, result, rollback path, next checkpoint
```
