# Safe Agent Box Readback Template

Use after an AI agent acts inside a box.

```text
Box name:

Goal:

Permission level used:

Files read:

Files changed:

Commands run:

Dependencies touched:

Secrets touched:
[should usually be none]

Tests/checks run:

Result:

Remaining issues:

Known debt:

Rollback path:

Next checkpoint:

Human Gate needed now:
```

## Readback verdict

```text
PASS:
PARTIAL:
FAIL:
NEEDS_HUMAN_REVIEW:
```
